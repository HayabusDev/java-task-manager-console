# Java Task Manager (Console)

## Overview
- This is a console-based task management application built with Java.
- Users can register, view, and complete tasks via a simple menu interface.

## Features
- Register new tasks
- Display task list
- Mark tasks as completed
- Input validation

## How to Run
1. Compile all Java files in the src directory.
2. Run Main.java.
3. Operate the application from the console menu.

## Learning Points
- MVC-style architecture (Controller / Service / Repository)
- Separation of concerns
- Basic input validation
- Exception handling and boolean-based error control
